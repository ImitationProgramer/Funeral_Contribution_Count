# 조문비 기록부 (Funeral Contribution Count)

장례식장에서 부의금(조문비)을 간편하게 기록하고 정리하는 **무료 웹앱**입니다.
설치나 회원가입 없이 브라우저에서 바로 사용할 수 있습니다.

**👉 바로 사용하기: https://imitationprogramer.github.io/Funeral_Contribution_Count/**

## 왜 만들었나요?

장례식장에서 부의금 봉투를 수기로 장부에 적으면, 장례가 끝난 뒤 유족별로 손님을 나누고 금액을 계산하는 데 시간이 오래 걸립니다.
이 앱은 접수하는 순간 데이터로 기록되어 **실시간 집계와 정산이 자동으로** 이루어집니다.

## 주요 기능

- **간편 접수** — 조문객 성함, 금액(빠른 금액 버튼 3만~50만), 어느 유족의 손님인지, 관계/소속을 입력. 접수 시간은 자동 기록
- **계좌 송금 지원** — 현금 부의금 외에 계좌 송금도 기록 가능. 금액 미확인 시 비워두고 입금 확인 후 수정
- **실시간 집계** — 총액·건수, 유족별 소계가 입력 즉시 자동 계산
- **유족 명단 자유 설정** — 고인 성함과 상주·자녀·손주 등 유족 명단을 상황에 맞게 추가/삭제
- **손님별 그룹 보기** — 전체 시간순 목록과 유족별 그룹 목록 전환, 성함·관계 검색, 페이지 나눔(15명 단위)
- **엑셀 내보내기** — 유족별로 그룹핑된 .xlsx 파일 생성 (그룹 머리글, 소계, 총 합계 포함). 한글 깨짐 없음
- **데이터 보호** — 브라우저에 자동 저장되어 창을 닫아도 유지. 원하면 내 컴퓨터 폴더에 JSON 파일로 백업 연결 가능 (Chrome/Edge)

## 사용 방법

1. [앱 열기](https://imitationprogramer.github.io/Funeral_Contribution_Count/) — 처음 열면 설정 창이 나타납니다
2. 고인 성함과 유족 명단(성함 + 관계)을 입력하고 저장
3. 조문객이 올 때마다 성함·금액·손님 측을 입력하고 **접수 등록**
4. 장례가 끝나면 **엑셀 내보내기**로 정산 자료를 내려받기

여러 날 진행되는 장례도 같은 기기·같은 브라우저로 열면 데이터가 그대로 유지됩니다.
더 안전하게 보관하려면 상단의 **📁 폴더 파일에 연결**을 눌러 JSON 파일로 자동 백업하세요.

## 개인정보

입력한 모든 데이터는 **사용자의 브라우저(기기)에만 저장**됩니다.
서버로 전송되지 않으며, 개발자를 포함한 누구도 데이터를 볼 수 없습니다.

## 권장 환경

- **Chrome, Edge** (권장) — 모든 기능 사용 가능
- Safari, Firefox — 사용 가능하나 "폴더 파일 연결" 백업 기능은 지원되지 않음 (브라우저 저장 + 엑셀 내보내기는 정상 동작)

## 기술

별도 서버나 데이터베이스 없이 동작하는 단일 HTML 파일입니다.
엑셀(.xlsx) 생성도 외부 라이브러리 없이 브라우저에서 직접 처리하므로 오프라인에서도 동작합니다.
